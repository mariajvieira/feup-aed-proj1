Function to add a given Student to UC without inputs from user.

No inputs from the user enables the use of the function to switch a UC of the user. The addition of the UC depends on various conditions: the class of the UC must have vacancy (no more than 26 students); the lectures of the new UC cannot overlap any TP or PL lecture in the schedule of the student; this change must maintain or improve balance of class occupation; Function complexity: O(n + m): where 'n' is the number of Student objects and 'm' is the number of UC objects.
